О системе

Система управления аэропортом

Данное приложение разработано в рамках курсового проекта по дисциплине "Разработка веб-приложений на Java с использованием Spring Boot".

Основные функции:
  • Управление рейсами (создание, редактирование, удаление)
  • Бронирование билетов пассажирами
  • Онлайн-регистрация на рейсы
  • Управление посадкой пассажиров
  • Паспортный и таможенный контроль
  • Управление пользователями и ролями
Технологии:
  • Backend: Java 21, Spring Boot 4, Spring Security, Spring Data JPA
  • Frontend: Thymeleaf, Bootstrap 5, JavaScript
  • База данных: PostgreSQL
  • Миграции: Liquibase
